This project is a static promotional website developed for a furniture company using HTML, CSS, and JavaScript. The website is designed to showcase products, share business information, and provide a simple, responsive interface for users to interact with.

Project Description
The website features a structured layout that highlights furniture collections and company branding. A responsive design ensures a smooth user experience across different devices, while the "Contact Us" section allows visitors to easily reach out. 
This project was developed to strengthen front-end development skills and practice building real-world business websites.

Technologies Used
HTML
CSS
JavaScript

Objective
To create a visually appealing, responsive, and user-friendly website that serves as an online promotional tool for a business. The project also aimed to enhance practical knowledge in web layout design and front-end coding standards.
